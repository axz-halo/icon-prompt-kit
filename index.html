재현율 팁

  1. 미드저니 sref: 같은 카테고리(과일만 등)로 채우면 주제가 오염됨 — 실측에서 돛단배가 배(pear)로 변형됐어요. 카테고리가 섞인 사물 3~4장(냄비·우산·수박조각·자동차처럼 광택 마크가 뚜렷한 것)으로 구성하고 --sw 250~300 · --chaos 0
  2. 미드저니: --v 7·--style raw가 코어에 내장, --s 50 이하 · 시리즈는 --seed 고정 · 출력은 흰 배경 PNG → 채택 컷은 Vectorizer.AI 벡터화 → 팔레트 스냅 → 샤인 마크 보정
  3. 힉스필드: 이미지 레퍼런스 1장 첨부, 네거티브 필드에 outline, stroke, gradient, shadow, 3d, texture, text
  4. hex는 방향 지시 — 최종 납품은 벡터 리드로우(트레이싱) 전제

테마 A 정밀 가이드 — 75종 SVG 실측 (2026-07-16)

  • 색: 아이콘당 고유색 평균 2.5개(중앙값 2, 최대 5) — 2~3색이 규격
  • 도형 수: 평균 4.4개(범위 1~9) — 극단적 미니멀. 10개 넘으면 세트 이탈
  • 도형 유형: path 93%(307/330) · circle 20 · rect 3 — 직사각형을 거의 쓰지 않고 둥근 커스텀 패스로 그림
  • 디테일 처리: 광택·씨앗은 stroke 속성이 아니라 둥근 캡 모양의 '채워진 패스' (linecap 사용 0건)
  • 색 위계: #EBEBEB 43/75(하이라이트 겸 흰색 대용) → #8BC02F 31 → #F2C500 22 → #42A141·#05A9E3 12
  • 구도: 중앙 배치 + 45° 대각 기울임 빈번(수박조각·시나몬·밀대 등), 원본 viewBox 32×32

외부 툴 제작 의뢰 가이드 (무엇으로 뽑을까)

  1. 미드저니 = 무드 탐색·다양성용: V7 기본 · 카테고리 섞인 sref 3~4장 --sw 250~300 · --chaos 0 — 색은 근사치로만 나오므로(실측) 채택 컷은 벡터화+팔레트 스냅 후처리 전제
  2. 힉스필드: 실사 지향 모델이라 플랫 벡터 재현율이 낮은 편 — 레퍼런스 이미지 첨부 필수 + 네거티브에 outline, gradient, shadow, 3d, photo, text
  3. AI = 프롬프트 가이드 생성: 주제를 입력하면 75종 실측 무드 스펙이 내장된 단일 완성 프롬프트(미드저니/힉스필드)를 작성해줘요 — sref 없이 한 개로 생성 가능. 모델: claude-sonnet-5 ⭐ / gpt-5.6-sol
  4. SVG 네이티브 외부 툴: Recraft(AI가 SVG 직접 생성) · Vectorizer.AI/일러스트레이터 Image Trace(미드저니 PNG→SVG 변환) — 플랫 단색이라 변환 품질 높음
  5. 의뢰 방법: 갤러리에서 유사 주제 카드 [프롬프트 복사] → 주제부(색·실루엣·디테일)만 교체해 전달 → 결과물은 아래 QA 6항목으로 검수
